Models

Summary

Models are the heart of the shopify-cart library. They contain the interactive data as well as a large part of the logic surrounding it, and provide a basic set of functionality for managing changes to the shopping cart.

Base

All of the models descend from ShopifyCart.Models.Base providing the following variables and methods:

  • attributes

    A hash that contains all of the model's properties. This variable should never be accessed directly. Instead use the get or set methods.

  • defaults

    A hash that contains all of the default properties for a new model.

  • change(model)

    This is triggered when an attribute in the model changes, and can be called to manually update anything bound to the model's changed event.

  • events

    So far there's only one event that's being used by shopify-cart: The changed event. Bind to this event to be notified of any changes to the model's attributes.

  • fetch()

    Call this method to fetch the shopping cart from the Shopify store.

  • get(name)

    Gets an attribute of the model.

  • save()

    Saves the model's data on the server.

  • set(name, value)

    Sets an attribute of the model.

  • toJSON()

    Converts the model's attributes to JSON.

  • url(method)

    Override this method to provide urls for each method. So far only GET and POST are being used.
